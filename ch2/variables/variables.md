# Переменные в Golang

Объявление ***var*** создает переменную определенного типа, назначает ей имя и присваивает начальное значение.
Общий вид:
```Golang
var name type = expression
```

Часть ***type*** или ***= expression*** может быть опущена, но не обе вместе:
```Golang
var name = expression
```
Если не указан тип переменной, то он определяется из инициализирующего выражения.

```Golang
var name type
```
Если опущено выражение, то начальным значением является нулевое значение для данного типа. Например **0** - для *чисел*, **false** - для *булевых* переменных, **""** - для *строк* и **nil** - для *интерфейсов* и *ссылочных* типов.

В Go не существует такого понятия, как неинициализированная переменная!

В одном объявлении можно объявлять и инициализировать несколько переменных. Пропущенный тип позволит объявлять несколько переменных разных типов.

```Golang
var i, j, k int // int, int, int
var b, f, s = true, 2.3, "four" // bool, float64, string
```

Переменные уровня пакета инициализируются до начала выполнения функции main, а локальные переменные инициализируются - когда в процессе выполнения функции встречается их объявление.

Так же есть переменные, которые инициализируются только тогда, когда вызываемая функция возвращает два значения, например:

```Golang
var f, err = os.Open(name) // os.Open возвращает файл и ошибку
```
