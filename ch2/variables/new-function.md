# Функция **new**

Встроенная функция ```new``` - это еще один способ создания переменной. Выражение ```new(T)``` создает неименованную переменную типа T, инициализирует ее нулевым значением типа T и возвращает ее адрес, который представляет собой значение типа *T. 

```Golang
p := new(int)
fmt.Println(*p) // 0
*p = 2
fmt.Println(*p) // 2 
```

Стоит отметить, что переменная, созданная с помощью ```new``` ничем не отличается от локальной переменной, у которой берется адрес, за исключением того, что нет необходимости придумывать и объявлять ее имя. Она является лишь синтаксическим удобством, например две данные функции имеют идентичное поведение:

```Golang
func newInt() *int {
    return new(int)
}
```

```Golang
func newInt() *int {
    var dummy int
    return &dummy
}
```

Кажыдй вызов new возвращает отличную от предыдущих переменную с уникальным адресом:

```Golang
p := new(int)
q := new(int)
fmt.Println(p == q) // false
```
Но существует исключение: две переменные, тип которых не несет никакой информации, а потому имеющие нулевой размер, такие как struct{} или [0]int, могут, в зависимости от реализации, иметь один и тот же адрес.

Так же функцию ```new``` можно переопределить, например:
```Golang
func delta(old, new int) int {return - old}
```
