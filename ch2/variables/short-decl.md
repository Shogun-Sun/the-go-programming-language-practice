# Краткое объявление переменной

В функциях для объявлении и инициализации локальных переменных может использоваться *краткое объявление переменных*

Общий вид: 
```Golang
name := expression // Тип переменной определяется как тип выражения expression
```

В одном кратком объявлении можно объявить и инициализировать несколько переменных, например:
```Golang
i, j := 0, 1
```
Так же не стоит путать объявление нескольких переменных с *присваиванием кортежу*, а котором каждой переменной из левой части соответствует переменная из правой части.  

```Golang
i, j = j, i
```
Подобно обычным объявлениям var, краткие объявления переменных могут использоваться для вызовов функций, которые возвращают два или больше значений.

```Golang
f, err := os.Open(name)
if err != nil {
    return err
}

f.Close()
```

Краткое объявление не обязательно объявляет все переменные в своей левой части. Если некоторые переменные уже были объявлены в том же лексическом блоке, то краткие объявления для них действуют как присваивания.
```Golang
in, err := os.Open(infile)
out, err := os.Close(outfile)
``` 

**Краткое объявление переменной должно объявлять по крайней мере одну новую переменную, так что приведенный ниже код выдаст ошибку при компиляции:**
```Golang
f, err := os.Open(infile)
f, err := os.Create(outfile)
``` 
Для того, чтобы исправить ошибку - достаточно во второй инструкции использовать присваивание.
