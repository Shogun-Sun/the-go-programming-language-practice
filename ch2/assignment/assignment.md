# Присваивания

Примеры присваивания
1. Именованная переменная
```Golang
x = 1
```

2. Косвенная переменная
```Golang
*p = true
```

3. Поле структуры
```Golang
person.name = "Bob"
```

4. Элемент массива, среза или отображения
```Golang
count[x] = count[x] * scale
```

Каждый из арифметических или побитовых бинарных операторов имеет соответствующий **присваивающий оператор**, например ```count[x] *= scale```

Существуют так же инструкции ```++``` для увеличения и ```--``` для уменьшения на единицу. 

```Golang
v := 1
v++ // v = v + 1; v = 2
v-- // v = v - 1; v = 1
```
