# Присваивание кортежу

**Присваивание кортежу** позволяет выполнять присваивание значений нескольким переменным одновременно. 

```Golang
x, y = y, x
a[i], a[j] = a[j], a[i]
```
Прежде чем любая переменная из левой части получит новое значение, вычисляются выражения из правой части.

Пример вычисления числа Фибоначчи с использованием **присваивания кортежу**
```Golang
func fib(n int) int {
    x, y := 0, 1
    for i := 0; i < n; i++ {
        x, y = y, x + y
    }
    return x
}
```

Так же, **присваивание кортежу** часто используют, когда вызывают функцию, возвращающую несколько значений. 
```Golang
f, err = os.Open("filename.txt")
```

Если же какая-то из переменных не нужна, можно просто присвоить ее пустому идентификатору.
```Golang
_, err = io.Copy(dst, src)
```
