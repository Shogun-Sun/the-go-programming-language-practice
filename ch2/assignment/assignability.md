# Присваиваемость

В программе существуют места, в которых присваивание происходит **неявно**, например вызов функции неявно присваиванет значения аргументов соответствующим переменным параметров; литеральное выражение для составного типа, такое, как срез:
```Golang
medals := []string{"gold", "silver", "bronze"}
```
Каждый элмент присваивается неявно, как если бы его записали так:
```Golang
medals[0] = "gold"
medals[1] = "silver"
medals[2] = "bronze"
```

Любое присваивание, явное или неявное, всегда корректно, если левая и правая части имеют один и тот же тип.
