# Импорт

В Go каждый пакет идентифицируется уникальной строкой, которая называется путём его импорта. Он обозначает каталог, сожержащий один или несколько исходных файлов Go, совместно образующих пакет. 

```Golang
import "the-go-programming-language-practice/ch2/packages-and-files/converter/tempconv"
```

Импорт пакета, без последующего его использования считается ошибкой.

Для более комфортной работы с импортами существует инструмент: ```golang.org/x/tools/cmd/goimports``` - он автоматически добавляет и удаляет пакеты из объявлений импорта. Чаще всего современные редакторы уже настроены на его использование при каждом сохранении файла.
