# Объявление типов
Тип переменной или выражения определяет характеристики значений, которые он моет принимать (количество битов, количество элементов), операции, которые могут быть над ними выполнены, связанные с ними методы.

Объявление ```type``` определяет новый именованный тип, который имеет тот же базовый тип, что и существующий. 

```Golang
type имя базовый тип
```
*Объявления типов* чаще всего используют на уровне паета.

Пример использования в [tempconv0](./tempconv0/main.go). Celsius и Fahrenheit созданы для двух шкал темпе­ратуры. Несмотря на то что оба они имеют один и тот же базовый тип, *float64*, сами они являются разными типами, а потому не могут сравниваться или объеди­няться в арифметических выражениях. 

Для каждого типа **T** имеется соответствующая операция преобразования **T(x)**, которая приводит значение x к типу **T**. Такое преобразование разрешено, если оба они имеют один и тот же базовый тип или если оба они являются неименованными указателями на переменные одного и того же базового типа.

Операторы сравнения, такие как **==** и **<>** могут использоваться для сравнения занчения именованного типас другим значением того же типа или со значением неименованного типа с тем же базовым типом. Но два значения различных именованных типов сравнивать непосредственно нельзя. 

```Golang
var c Celsius
vat f Fahrenheit

fmt.Println(c == 0) // true
fmt.Println(f >= 0) // true
fmt.Println(c == f) // Выдаст ошибку при компиляции о несоответствии типов.
fmt.Println(c == Celsius(f)) // true
```
