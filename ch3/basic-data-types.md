# Фундаментальные типы данных

На уровне железа вся память — это биты, объединенные в слова (фиксированные пакеты битов). Компьютер интерпретирует их как числа, адреса или битовые множества.

Язые ```Go``` соединяет два уровня:

1. Низкоуровневый: типы, которые напрямую соответствуют возможностям аппаратного обеспечения (числа).

2. Высокоуровневый: инструменты для построения сложных структур данных, удобных для человека (пакеты, пиксели, портфолио).

Категории типов в Go:

1. **Фундаментальные / Базовые типы (Basic types)**: 
    * Числа (целые, с плавающей точкой, комплексные);
    * Строки (string); 
    * Булевые значения (bool);
    * Простые атомарные значения.
2. **Составные / Агрегированные типы (Aggregate types)**:
    * Массивы (arrays);
    * Структуры (structs);
    * Строятся путём объединения более простых типов в одну общую структуру.
3. **Ссылочные типы (Reference types)**:
    * Указатели;
    * Срезы (слайсы);
    * Отображения (мапы);
    * Функции;
    * Каналы;
    * Они ссылаются на переменные или состояние программы косвенно.
4. **Типы интерфейсов (Interface types)**:
    * Особый тип данных, который используется для описания контрактов и абстрактного поведения.
