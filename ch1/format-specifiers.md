# Спецификаторы формата в GO

| Спецификатор | Тип данных                                                        |
| ------------ | ----------------------------------------------------------------- |
| %d, %i       | Десятичное целое                                                  |
| %x, %o, %b   | Целое в шестнадцатиричном, восьмиричном и двоичным представлениях |
| %f, %g, %e   | Числа с плавающей точкой                                          |
| %t           | Булевое значение: true или false                                  |
| %c           | Руна (символ Unicode)                                             |
| %s           | Строка                                                            |
| %v           | Любое значение в естественном формате                             |
| %T           | Тип любого значения                                               |
| %%           | Символ проента                                                    |
